# react-secuuth

> Made with create-react-library

## Install

```bash
npm install --save react-secuuth
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'react-secuuth'
import 'react-secuuth/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
